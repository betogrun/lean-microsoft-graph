0.4.0 - 2025-01-07

Added

Changed

  • Upgrade dependencies: faraday, mocha and minitest.

0.2.4 - 2024-05-23

Changed

  • Upgrade dependencies: mocha and minitest.

Added

0.2.0 - 2023-11-04

Added
